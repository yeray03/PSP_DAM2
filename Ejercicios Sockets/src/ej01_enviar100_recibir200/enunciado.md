# Ejercicio 01: Enviar 100 - Recibir 200

## Descripción

Crea un programa Cliente que envíe al servidor un número 100. 

El Servidor deberá responder con un 200.

## Requisitos

- **Cliente**: 
  - Envía el número 100 al servidor
  - Debe esperar la respuesta del servidor
  
- **Servidor**: 
  - No debe terminar nunca
  - Debe aceptar todas las peticiones sin importar el número de clientes
  - Responde con el número 200
